PHÂN LOẠI THẺ

THẺ BLOCK

  1. hx : định dạng tiêu đề
  2. p : định dạng đoạn văn
  3. div : phân chia khu vực
  4. ol,li-ul,li : danh sách theo thứ tự (hoặc ko )
  5. nav : chứa thanh điều hướng
  6. figure : để xài figcaption với điều chỉnh width height anh
  7. pre : thể hiện nội dung
  8. main : chứa nội dung chính
  9. header : đầu trang web
  10. section : khu vực chứa nội dung (thân trang web)
  11. article : nội dung các phần (con section)
  12. aside : bổ nghĩa cho article , quảng cáo
  13. input : tạo ô nhập
  14. button : nút
  15. textarea : tạo ô nhập to
  16. legend : siêu nhấn mạnh
  17. footer : cuối web

THẺ INLINE

  1. br : xuống dòng
  2. i : in chéo
  3. b : in đậm
  4. hr : gạch ngang lớn
  5. u : gạch chân
  6. sub : hoá trị
  7. sup : mũ
  8. strong : xài thay cho b
  9. em : xài thay thẻ i
  10. span : nối thẻ inline
  11. a : dẫn link web
  12. label : form chữ
  13. img : ảnh

  3. target : chuyển trang , ghi đè lên trang cũ
  4. alt : hiện nội dung trong alt ( bth là khi đường dẫn lôi)
